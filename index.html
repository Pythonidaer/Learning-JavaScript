Things I've learned in JavaScript

From videos 1-10:

  • String Interpolation: done with back ticks ``, embeds variables with ${}.
  • String Concatenation: the process of joining two or more strings into a bigger one.
  • Strings: " ", ' ', ` `, datatypes used to represetn text rather than, say, numbers.
  • Template Strings: template literals that allow embedded expressions and use of syntactic sugar.
  • A newline usually implies a semicolon.
  • Arrays: stores list items you'll want to use in your code.
  • [... var] the spread operator allows you to create a new array that won't copy over its original.
  • How to comment out multiple lines of code: highlight with cursor, hit Ctrl + /.
  • Identity function: returns whatever is passed into it.
  • Objects: unordered collections of related data
  • Key-Value Pairs: a property
  • Keys: strings, a property name
  • Values: can be anything
  • Dot operator: used to reference individual members of a class, structure, or union
  • Primitive: values containing only one thing.
  • Datatypes: particular items, as defined by the values it can take, language used, or operations that can be performed on it.

Here are some methods and other things I have learned:

  • .trim() function returns a new string without any of the leading or the trailing white spaces.
  • .length property returns the length of a string in number of characters; an empty is 0.
  • .toUpperCase() function converts the entire string to UPPER CASE.
  • .toLowerCase() function converts the entire string to lower case.
  • .push() method adds new items to the end of an array, and returns the new length.
  • .pop() removes the last item
  • .map() allows you to make a copy of an array
  • .hasOwnProperty() checks for property, prints out a Boolean value.
  • Console.log() function used to prin tany kind of variables defined before in it or to print a message.
  • Object.keys() method returns an array of the given object's property names, same order as a standard loop.
  • Object.values() method returns an array whose elements are the enumerable property values found on an object.
  • Object.assign({}, var) method is used for cloning an object; returns all own enurable properties from one or more source objects.
  • parseInt() function parses a strin gand returns an integer.
  • parseFloat() an inbuilt function which is used to accept the strin gand convert it into a floating point number.
  • Boolean() function returns the boolean value of a variable, can work on truthy and falsy values.
